# 🚀 Cisco Network Simulation Project

## 🧠 Project Overview

This project showcases a **network simulation built using Cisco Packet Tracer**, designed to demonstrate practical understanding of networking fundamentals and real-world configuration scenarios.

The simulation includes structured topology design, device configuration, and connectivity validation across multiple network segments.

---

## 🏗️ Network Topology

The network is designed to represent a small-to-medium scale infrastructure consisting of:

* Routers for inter-network communication
* Switches for local network distribution
* End devices (PCs) for connectivity testing

Each device is configured to ensure proper communication across the network.

---

## ⚙️ Key Configurations

This project covers:

### 🔹 Routing

* Static Routing / Dynamic Routing (depending on setup)
* Inter-network communication between different subnets

### 🔹 Switching

* Basic Switch Configuration
* VLAN implementation (if applied)

### 🔹 IP Addressing

* Structured IP addressing scheme
* Subnetting for efficient network segmentation

### 🔹 Connectivity Testing

* Ping tests between devices
* End-to-end communication validation

---

## 🛠️ Tools & Technologies

* **Cisco Packet Tracer**
* CLI-based configuration (Cisco IOS commands)

---

## 📂 Project File

* `F2.1_Denis Lapianso_Cisco.pkt` → Main simulation file

## ▶️ How to Run

1. Open Cisco Packet Tracer
2. Load the `.pkt` file
3. Navigate through the topology
4. Access devices via CLI
5. Run simulation mode or use ping tests to verify connectivity
